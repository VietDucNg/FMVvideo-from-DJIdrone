1. Get DJI flight log file

1.1. DJI Fly

With video recorded by DJI Fly app, the corresponding flight log file can be found here:

  • Android\data\dji.go.v5\files\FlightRecord (for Android devices)
  • DJI\dji.go.v5\files\FlightRecord (for iOS devices)
